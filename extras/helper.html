Legend mod for agar.io multiplayer action Chrome browser game (combines all).
This Mod is BIG!

Why Legend? (Your last mod)

1. This mod offers players all ogario feautures that include rendering of the game canvas
2. Is ALIVE! Auto updates (occasionally may need some manual updates)
3. FFA/EXP/TEAM/PARTY integrated chat and minimap
4. Unlimited FPS - Unlocked, gameplay is quicker than Vanilla
5. Handlers for SIP (compatible with other mods/vanilla), party tokens, IP
6. Handlers for message commands, settings, locations, modes, etc
7. Copy Tokens/SIP (Tokens for party, SIP locked servers for other modes),
IP(search), leaderboard from the leaderboard hud
8. Leaderboard and Minimap huds, and many settings
9. Textures are very editable, make your own settings

How to find the same server?

1. SIP-TOKENS: Under the leaderboard click COPY->TOKEN. This copies the TOKEN if mode is party, or the SIP locked server for other modes
SIPs created are compatible with other mods and vanilla but also provide other information. (Use on URL)
Joining with SIP provides more FPS than joining with Connector

2. Copy the IP or party token from connector, send to your friend and Connect.

3. You can also use Server WS or Server token by clicking on utility button which is on the left side the region combo box.

Container Tabs

HOME: Used for changing name, password, region, mode,or to login, play, spectate, and more
USER: Agario stuff, quests, skins, logout, etc
API: Tools, the light blue buttons are suggested to be opened, the purple are optional
SETTINGS: Canvas rendering tools, very helpful for game!
THEME: Settings about the interface of the mod, including 6 mini-tabs
HOTKEYS: Command hotkeys for keypress events
YOUTUBE AND SOUNDS: Youtube with url of your choice, can autostart when game starts if chosen

More Stuff

Hide/Show everything and Timer calculator have been requested by youtubers/quest players
Change your minimap/leaderboard/teamboard/main canvas with a picture url or customise message icons&Youtube videos e.g http://i.imgur.com/RVBi3T1.gif, or change your chat position
On TOOLS tab, the Shortcuts button is suggested be active. You can use binoculars on specate mode (or press Q), use copy tools, youtube button or fullscreen
There is also a Chat button to hide/show to team, 6 message icons that you can send the default imgur icons or yours, and 6 message commands, mod gives direct commands to the script of your team player

Also you can find 2 side containers, the left is a small panel of things and the right is for custom profiles.
Import/export your own settings by clicking to tab SETTINGS->IMPORT/EXPORT SETTINGS-> Paste your code (and import) or export it
